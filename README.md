# Breathing
フェードインアウトにより深呼吸を促すGUIアプリ。左上画面端トリガーにより手軽に前面化／最小化を切り替えることができます。  

【基本操作】  
・batファイルをダブルクリックで起動できます  
・画面の左上端にマウスカーソルを寄せると前面化＆フェードの開始／終了を行います  
・画面の左下端にマウスカーソルを寄せると最小化します
